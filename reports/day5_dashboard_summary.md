# Day 5 - Dashboard Development Summary

## Objective

Develop an interactive Power BI dashboard to visualize mutual fund industry trends, fund performance, investor analytics, and SIP market insights using the cleaned datasets prepared during previous tasks.

## Work Completed

### Data Integration

* Imported all cleaned CSV datasets into Power BI.
* Verified successful data loading.
* Created relationships between tables using common fields such as `amfi_code` and `date`.

### Dashboard Pages

#### Page 1 – Industry Overview

* KPI Cards:

  * Total Industry AUM
  * Monthly SIP Inflows
  * Total Folios
  * Total Schemes
* Industry AUM Trend (2022–2025)
* AUM by Fund House

#### Page 2 – Fund Performance

* Return vs Risk Scatter Plot
* Fund Performance Scorecard Table
* NAV Trend Visualization
* Interactive slicers for Fund House, Category and Plan

#### Page 3 – Investor Analytics

* State-wise Investment Distribution
* Payment Mode Distribution
* Average Investment by Age Group
* Monthly Transaction Trend
* Interactive slicers for State, Age Group and City Tier

#### Page 4 – SIP & Market Trends

* SIP Inflow vs Nifty 50 Combined Chart
* Category-wise Net Inflow Analysis
* Top 5 Categories by Net Inflow
* Category slicer

### Dashboard Features

* Added drill-through navigation for fund details.
* Enabled interactive filtering using slicers.
* Applied dashboard formatting and chart titles.
* Verified default Power BI tooltips.

## Deliverables

* bluestock_mf_dashboard.pbix
* Dashboard.pdf
* Dashboard page screenshots (PNG)

## Outcome

Successfully developed a four-page interactive Power BI dashboard providing insights into mutual fund industry performance, investor behaviour, SIP trends, and market analysis.
